## Investments bot

Setup commands:

You will need docker and Make command installed.

### Setup

#### Build:

```
make build
```

#### Start:

```
make build up
```

#### Run fixtures:

```
make build up
```

After that start frontend and edit .env - add your backend url to it.

### Useful things

#### Check state:

```
make build ps
```

#### Go into container:

```
make ssh
```
